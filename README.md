Task 1 - LANDING PAGE FOODMUNCH
– Responsive Food Store Website
Tagline: Get Delicious Food at Anytime.

Project Description:
FoodMunch is a responsive food store website developed using HTML, Tailwind CSS, and CSS. It allows users to explore food menus, healthy food options, delivery services, and special offers through a simple and attractive interface.

Technologies Used:
HTML5
Tailwind CSS
CSS
Google Fonts (Roboto)

Key Features:
Hero Banner Section
Why Choose Us Section
Explore Menu Section
Healthy Food Section
Delivery & Payment Information
Customer Gift Section
Follow Us & Footer
Responsive Design

Project Structure:
index.html – Contains the website structure and content.
style.css – Applies custom styles and fonts.
External Assets – Includes images and icons used in the project.

FoodMunch is a modern and user-friendly frontend project that demonstrates responsive web design and UI development skills. It is an excellent project for beginners to practice building attractive websites.

Outcome:
Successfully developed a fully responsive food store landing page using HTML, Tailwind CSS, and CSS. The project improved my skills in responsive web design, layout creation, and building attractive user interfaces.
