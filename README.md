Task 1 - Landing Page
FoodMunch – Responsive Food Store Website
Tagline: Get Delicious Food at Anytime.

Project Description:
FoodMunch is a responsive food store website developed using HTML, Tailwind CSS, and CSS. It allows users to explore food menus, healthy food options, delivery services, and special offers through a simple and attractive interface.

Technologies Used:
HTML5
Tailwind CSS
CSS
Google Fonts (Roboto)

Key Features:
Hero Banner Section
Why Choose Us Section
Explore Menu Section
Healthy Food Section
Delivery & Payment Information
Customer Gift Section
Follow Us & Footer
Responsive Design

Project Structure:
index.html – Contains the website structure and content.
style.css – Applies custom styles and fonts.
External Assets – Includes images and icons used in the project.

FoodMunch is a modern and user-friendly frontend project that demonstrates responsive web design and UI development skills. It is an excellent project for beginners to practice building attractive websites.

Task 3 - Temperature Converter
Temperature Converter – Documentation

Description:
Temperature Converter is a simple and user-friendly web application that converts temperature values between Celsius, Fahrenheit, and Kelvin. It provides quick and accurate results through an attractive and responsive interface.

Technologies Used:

HTML

CSS

JavaScript


Features:

Converts temperatures between Celsius, Fahrenheit, and Kelvin.

Validates user input before conversion.

Displays instant results with two decimal places.

Modern glassmorphism design with responsive layout.


HTML is used to create the structure of the application, including input fields, dropdown menus, and result sections. CSS enhances the appearance with gradient backgrounds, blur effects, and hover animations. JavaScript performs the temperature conversion calculations and updates the converted result dynamically.

🛠 Technologies Used:

HTML5

CSS3

JavaScript
